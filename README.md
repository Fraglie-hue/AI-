# Edge Note
### 基于前端云的文章摘要与笔记工具

##项目介绍
Edge Note是一个基于**阿里云ESA Pages** 部署的前端应用，用于对长文本进行快速摘要与重点提取。
用户只需粘贴文章内容，即可生成摘要，并通过**摘要与原文高亮对比**的方式，快速定位关键信息在原文中的位置，降低阅读和理解成本。
项目无需后端、无需登录，数据仅保存在本地，适合学习笔记、技术文档整理等即开即用场景。

---

## 核心功能
- ✍️ 本地笔记管理（新建 / 删除 / 切换）
- 🧠 多模式摘要生成
  - 智能摘要
  - 首段摘要
  - 定长摘要  
- 🔍 摘要与原文高亮对比
- 🌙 深色 / 浅色主题切换
- 📦 支持导出 Markdown / TXT

--- 

## 应用场景
- 📚 学生整理学习资料与课堂笔记  
- 💻 开发者阅读和整理技术文档  
- ✏️ 内容创作者快速提炼文章核心要点  

项目部署完成后即可直接使用，无需额外配置。

---

## 技术方案
- Vue 3 + Composition API
- Vite 构建
- localStorage 数据持久化
- 纯前端静态部署，适合边缘计算场景
